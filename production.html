Ветровое стекло
Алюминиевая рама с остеклением травмобезопасным небьющимся акриловым стеклом. Рама окрашена эпоксиполиэфирной краской (базовый цвет серый), снабжена открываемой центральной секцией с «задрайкой» и резиновым ограничителем. В комплекте – уплотнитель для установки рамы.

Багажный ящик для снегоходов:  Yamaha “Viking” и Arctic cat 570

Надежный, удобный и очень вместительный багажный ящик с 2 – мя держателями для ледобуров.  Крышка с двумя защелками выполняет функцию удобного стола.

Стеклопластик. Базовые цвета: черный, белый.

Yamaha “Viking”

Вместительный кофр объемом 105 литров. В верхней части на крышке кофра расположен большой карман предназначенный для хранения стояночного чехла (чехол приобретается отдельно). Крепление к багажнику при помощи застежек с липкой лентой. Молнии усиленные 10 мм.

Внутреннее пространство сумки делится твердыми съемными перегородками на 2 части. Кофр хорошо держит форму не деформируется со временем. Не дает промерзать вещам внутри. В нижней части удобные ручки для переноса сумки.

Стоимость: по запросу
Arctic cat 570

Кофр для утилитарных снегоходов Arctic Cat на платформе XT. Багажная сумка -кофр для снегохода Arctic Cat изготовлен из морозостойкой ткани полиэстер 900Х900 , дно выполнено из непромокаемой ткани ПВХ . Стенки, дно и верхняя крышка для жесткости армированы вставками из пенополиэтилена (ППЭ).Кофр прочно фиксируется в багажнике снегохода с помощью ремней и пряжек. Для переноски кофра имеются удобные регулируемые “рюкзачные “ лямки.

В верхней части кофра имеется карман. Верхняя крышка закрываеться с помощью застежек фастекс 30 мм в кол-ве четырех шт.. На боковых стенках имееться по карману ,на передней части кофра есть большой объемный карман.На всех карманах застежка молния . Кофр имеет защитный чехол который убирается в карман находящейся под большим карманом на лицевой стороне. Внутренний объем кофра разделен на два отделения с помощью съемной перегородки.Также при помощи ремней на верхней крышке можно прикрепить негабаритный груз.
